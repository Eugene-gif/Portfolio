# [Portfolio](https://eugene-gif.github.io/Portfolio/) - мой сайт портфолио.


<h3>Описание:</h3>

- Присутствует адаптив.
- Весь сайт размещен на одной html-странице.
- Подключены iframe с других моих проектов.
- Используется чистый js. 
- Модульность.
- Методология БЭМ.
- Препроцессор SASS(Scss).
- Все изображения оптимизированы.
- svg объединены в один svg-sprite.
- Стили и скрипты скомпилированы и сжаты.
- Сборка проекта - Gulp.
