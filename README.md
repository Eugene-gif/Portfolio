# [Portfolio](https://eugene-gif.github.io/Portfolio/) - мое портфолио.


<h3>Описание:</h3>

- Присутствует адаптив от 320px.
- Используется чистый JS. 
- Проект написан по методологии БЭМ.
- Есть модульность.
- Использован препроцессор SASS(Scss).
- Все изображения сжаты и оптимизированы.
- Все SVG объединены в один SVG-sprite.
- Стили и скрипты скомпилированы и сжаты каждый в отдельном файле.
- Сборка проекта Gulp.
