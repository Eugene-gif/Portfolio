  • Вертикальный слайдер

    Описание проекта

    Вертикальный слайдер - это красочный адаптивный проект написанный на чистом CSS & HTML + JS. Весь проект реализован на одной HTML-странице, стили написаны в одном отдельном CSS-файле.   • Desire

    Описание проекта

    Desire - это 7 страничный адаптивный сайт дизайна мебели. При разработке использовалась библиотека JQuery. Сайт верстался по методологии БЭМ, соответственно большая часть стилей и скриптов разбиты на отдельные модули. В проекте использовался препроцессор Scss, все изображения оптимизированы, SVG объединены в один SVG-спрайт. Стили и скрипты скомпилированы и сжаты каждый в отдельном файле. Для сборки проекта использовался Gulp.
